/// 01 - Project Description

This is a website for playing various sounds to fall asleep to. It has three audio players (Rain, White-Noise, & Fire). The sounds can be mixed and matched and their volumes independently adjusted. I use these types of apps often and wanted to create this one for some practice with web API's. You can view the repository for the project on GitHub here.
